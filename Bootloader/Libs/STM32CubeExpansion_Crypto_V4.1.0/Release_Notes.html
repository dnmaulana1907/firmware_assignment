Purpose

  • STM32 Cryptographic Firmware Library includes a set of cryptographic algorithms based on firmware implementation ready to use with all STM32 series. The main algorithms are :

    • Cipher algorithms : AES (modes CBC, CCM, CFB, CTR, ECB, GCM, OFB, XTS, Keywrap), SM4, Chacha20-Poly1305

    • Hash methods : SHA-1, SHA-2, SHA-3, SM3, SHAKE

    • Message Authentication Code : CMAC, HMAC, KMAC

    • ECC : ECDH, ECDSA, EdDSA, SM2

    • RSA : PKCS#1 v1.5, PKCS#1 v2.2, Chinese remainder theorem (CRT) key representation

    • DRBG

  • Main Cryptographic algorithms and their individual components are validated by the NIST, following ACVP program. Refer to certification page on Wiki pages for Cryptographic Library

  • STM32 Cryptographic Firmware Library provides a rich set of examples covering the main features of main cryptographic algorithm.

STM32 Cryptographic package provide a set of examples covering all available algorithms for the most common development tools. Examples are available in source file format and provided with preconfigured projects for the supported boards. Please refer to readme.txt of each example for more details about its purpose.

STM32 Cryptographic Firmware Library is located in middlewares level and support all STM32 series.

Reference documentation:

  • DB2660: Databrief of STM32 Cryptographic library for STM32Cube
  • Wiki pages for Cryptographic Library: Overview, getting started, secure usage, performances, certification and migration guide.

Main Changes

Major evolution of Cryptographic libraries for STM32 Series. This version offers :

  • New features :

    • Cipher algorithm: SM4 (CBC, CFB, CTR, ECB, OFB)

    • HASH methods: SHA-3, SM3, SHAKE

    • Message Authentication Code: KMAC

    • Assymetric algorithm: SM2, RSA (v2.2, Chinese remainder theorem (CRT) key representation, Bellcore Attack protection)

  • ACVP certification of all libraries, for all Cortex-M architecture

  • New User interface, simpler to use. Helpers are provided to ease the migration of user application from V3.x to V4.x version

  • Better performances : optimised in memory size and in speed.

  • Cryptographic libraries are compliant with MISRA C:2012 standard

Contents

  • Cryptographic libraries for Cortex-M0/M0+, Cortex-M3, Cortex-M4, Cortex-M7 and Cortex-M33

  • Application projects to demonstate Cryptographic on STM32G0 Series, STM32G4 Series, STM32H7 Series, STM32L0 Series, STM32L1 Series, STM32L4 Series, STM32L5 Series, STM32WB Series, STM32WL Series

  • The exhaustive list of projects is provided in this table (STM32CubeProjectsList.html)

  • Applications are ported on 3 toolchains : STM32CubeIDE, IAR and Keil
